# AI Mixing Assistant (V1.3)

## 🎧 Overview
AI-powered tool that analyzes your mix by comparing it with a reference track and provides intelligent feedback.

## 🚀 Features
- Mix vs Reference comparison
- Frequency-based issue detection
- Confidence & severity scoring
- Actionable suggestions (what to fix)
- Spectrum visualization (graph output)

## 🧠 How It Works
- Loads user and reference audio
- Performs frequency analysis (FFT)
- Detects imbalances in key frequency bands
- Generates human-readable insights
- Outputs a spectrum comparison graph

## 🛠️ Tech Stack
- Python
- Librosa
- NumPy
- Matplotlib

python src/main.py
